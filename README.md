# Nuxt Responsive App

* Use `ui.nuxt.com/getting-started` as a reference.
* To start the project:
  * set up a new Nuxt project
  * Add `@nuxt/ui` module
* In `app.vue` the basic layout is:
  * Header (Nav included)
  * Main
    * Sidebar
    * Content
* To be responsive for mobile screen:
  * A full screen Modal
    * Header (Narrow)
    * Nav
    * Sidebar
